# 1주차 학습 일지
이번 1주차 스터디에서는 HTML의 기본 구조와 웹페이지를 구성하는 핵심 태그들에 대해 학습하였다. 먼저 h1, h2와 같은 제목 태그를 통해 문서의 구조를 나누는 방법을 이해하였고, p 태그를 사용하여 문단을 구성하는 방법을 익혔다. 또한 a 태그를 활용하여 서로 다른 HTML 파일을 연결하는 하이퍼링크 기능을 직접 구현해보았다. 이를 통해 하나의 페이지가 아닌 여러 페이지로 구성된 웹사이트의 기본 원리를 이해할 수 있었다. 더불어 script 태그를 사용하여 console.log와 alert를 실행해보며 자바스크립트의 기본적인 동작도 경험하였다. 이번 학습을 통해 웹페이지의 기본적인 구조와 페이지 간 연결 방식에 대해 전반적으로 이해할 수 있었으며, 앞으로 더 다양한 기능을 추가할 수 있는 기초를 다질 수 있었다. 자기소개 페이지에 이미지는 넣어보지 못했는데, 다음 과제에서는 선택사항도 다 수행해보려는 능력을 길러야 겠다.
